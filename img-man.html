Image Manipulation

Image Manipulation is a primitive photoshop application allowing users to perform the following operations on an image loaded from their machine:

  • • Flip/Rotate
  • • Mirror Horizontally/Vertically
  • • Warm/Cool
  • • Blur/Sharpen

Users can then save the edited image as a jpeg/png on their machine for future use.

Created: 2015
Implemented with: Java
