PROJECTS

Orb It! - 2015

A version of the popular kids game Bop-it!. Uses numerous arduino libraries(sound,display,etc.) to create an interactive playing experience.

A primitive photoshop application that supports rotations, colour filters, mirror images, and flips. Ability to undo/redo and save edited images.
